# BMI Calculator

This app works for both iOS and Android, allowing one to calculate BMI using both the metric and imperial systems of measurement.
It also converts measurements correctly from system to system, and one can switch systems of measurementat anytime.

Finally, this app displays your BMI result along with an interpretation on a results screen, and allows users to recalculate as many times as they would like.
There are limits on the maximum and minimum height values, although these values are outside the smallest average height of a newborn and the largest height ever recroded.
